## ✨ 实现功能

### 1. 渡劫系统UI

#### 渡劫按钮（主游戏界面）
```html
<button id="btn-tribulation" class="btn-action btn-tribulation">
    <span class="action-icon">⚡</span>
    <span class="action-text">渡劫</span>
</button>
```

**显示条件**：
- 仅在筑基期Lv10时显示
- 带有发光动画特效（橙色光晕）
- 动态显示/隐藏

#### 境界提示区域
```html
<div id="realm-hint" class="realm-hint">
    <p>修炼进度：XX%，或渡劫提示</p>
</div>
```

**提示内容**：
- 正常修炼：显示经验百分比
- 可突破：显示突破提示
- 需要渡劫：显示渡劫警告
- 邪修限制：显示夺舍提示

#### 渡劫界面（独立屏幕）
```
┌─────────────────────────────┐
│      ⚡ 天劫降临             │
├─────────────────────────────┤
│ ⚠️ 渡劫有风险，失败将损失修为！│
│                             │
│ 当前境界：筑基期 Lv10       │
│ 下一境界：结丹期            │
│ 成功率：60%                 │
├─────────────────────────────┤
│ 渡劫结果                    │
│ ✅ 成功：晋升结丹期...      │
│ ❌ 失败：损失30%修为...     │
├─────────────────────────────┤
│ [开始渡劫]  [取消]          │
└─────────────────────────────┘
```

**特性**：
- 实时计算成功率（受幸运影响）
- 清晰显示成功/失败后果
- 确认后执行渡劫逻辑

### 4. JavaScript逻辑新增

#### UI.new Methods

**showSkillPanel()**
```javascript
- 获取可用技能列表
- 动态生成技能按钮
- 显示技能面板
```

**hideSkillPanel()**
```javascript
- 隐藏技能面板
```

**getAvailableSkills()**
```javascript
- 根据职业返回职业技能
- 如果控云天赋，添加6系法术
- 返回技能数组（含名称、消耗、action）
```

**useSkill(skill)**
```javascript
- 执行技能action
- 显示技能效果日志
- 更新敌人血条
- 检查胜利/继续战斗
- 触发怪物反击
```

**showTribulationScreen()**
```javascript
- 检查是否可以渡劫
- 计算成功率
- 更新界面数据
- 显示渡劫界面
```

**startTribulation()**
```javascript
- 调用Cultivation.attemptTribulation
- 显示成功/失败消息
- 延迟跳转回主界面
- 更新UI
```

**updateRealmHint()**
```javascript
- 获取境界提示文本
- 显示/隐藏提示框
- 显示/隐藏渡劫按钮
```

**updateMainGameUI() 增强**
```javascript
- 新增调用updateRealmHint()
- 自动显示境界提示
- 动态控制渡劫按钮
```
