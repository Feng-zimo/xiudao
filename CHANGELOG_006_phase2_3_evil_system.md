### 1. 魂魄收集系统 ☠️

#### 击杀获得魂魄
```javascript
EvilCultivation.gainSouls(player, monster)
```

**获得数量计算**：
```
基础：1-3个随机
境界加成：realmIndex * 0.5
示例：筑基期(1) → 1.5-4.5个，取整后1-4个
```

**自动触发**：
- 战斗胜利时自动调用
- 仅邪修阵营有效
- 实时显示在右侧面板

EvilCultivation.devourSouls(player, soulCount)

**效果**：
- 消耗1个魂魄

- 记录吞噬历史

**UI交互**：
- 点击"吞噬魂魄"按钮
- 实时显示魂魄数量
- 日志提示获得修为

### 3. 第二人格融合 ⚡

#### 融合机制
```javascript
EvilCultivation.fuseSecondPersonality(player)
```

**每次融合**：
- 消耗：50个魂魄
- 融合进度提升：10-20%（随机）
- 融合风险增加：+5%

**融合进度显示**：
```
当前：XX%
风险：XX%
```

#### 融合完成判定（100%进度）

**成功率计算**：
```
successChance = 1 - (risk / 100)
示例：风险20% → 80%成功率
     风险80% → 20%成功率
```

**融合成功**：
```
✅ 属性大幅提升：
   - 攻击 × 1.8
   - 防御 × 1.8
   - 生命 × 1.5
   - HP回满
✅ 清除第二人格数据
✅ 实力质变！
```

**融合失败**：
```
❌ 第一人格受损：
   - 攻击 × 0.7
   - 防御 × 0.7
✅ 重置融合进度为0
✅ 风险重置为20%
⚠️ 可以重新融合
```

**策略建议**：
- 风险低时尽快融合（成功率高）
- 风险高时谨慎尝试（可能前功尽弃）
- 平衡收益与风险

### 4. 夺舍重生系统 💀

#### 夺舍条件
```
✅ 必须是邪修阵营
✅ 必须已开启第二人格
✅ 目标怪物等级 ≤ 自身+2个大境界
```

#### 夺舍成功率
```javascript
baseChance = 50%
levelPenalty = (怪物等级 - 玩家等级) * 5%
finalChance = max(10%, baseChance - levelPenalty)
```

**示例**：
- 同级怪物：50%成功率
- 高3级怪物：35%成功率
- 高8级怪物：10%成功率（最低）

#### 夺舍成功
```javascript
EvilCultivation.completePossession(player, targetMonster)
```

**属性合并**：
```
新攻击 = 原攻击×0.5 + 怪物攻击
新防御 = 原防御×0.5 + 怪物防御
新生命 = (原生命 + 怪物HP) × 0.6
HP回满
```

**效果**：
- ☠️ 获得新肉体
- 保留50%原属性
- 获得怪物部分属性
- 记录夺舍历史

#### 夺舍失败
```javascript
EvilCultivation.failPossession(player)
```

**惩罚**：
- ❌ 损失50%当前修为
- ❌ HP降至1点
- 记录失败历史

**高风险高回报**：
- 成功：实力大增
- 失败：修为大跌
